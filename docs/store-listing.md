# VoxInk — Play Store Listing

## English (en-US)

### Title
VoxInk — AI Voice Keyboard

### Short Description (80 chars max)
AI voice keyboard with BYOK. Speak naturally, get polished text instantly.

### Full Description

VoxInk (語墨) is an open-source AI-powered voice keyboard for Android. Speak naturally and get clean, polished text — powered by your own API keys.

**Key Features:**

- Voice-to-text with Groq Whisper (world's fastest inference)
- Smart text refinement — removes filler words, fixes grammar, adds punctuation
- Dual result display — tap Original or Refined to insert
- Supports Traditional Chinese, English, Japanese, and auto-detect
- Mixed-language support (中英混合, etc.)
- Audio file transcription with export (TXT, SRT)
- BYOK model — use your own API keys, no subscription
- Privacy-first — no data collection, audio goes directly to your provider
- Material You dynamic theming

**How It Works:**

1. Enter your Groq API key (free at groq.com)
2. Enable VoxInk Voice in system keyboard settings
3. Switch to VoxInk in any app
4. Tap the mic button and speak
5. Tap the refined text to insert — done!

**Privacy:**

VoxInk uses a Bring Your Own Key (BYOK) model. Your audio is sent directly from your device to the API provider you choose. We never see your data.

**Open Source:**

VoxInk is fully open-source. Inspect the code, contribute, or build your own version.

語墨（VoxInk）是一款開源 AI 語音鍵盤。自然說話即可取得乾淨、精修的文字 —— 使用你自己的 API 金鑰。

**主要功能：**

- Groq Whisper 語音轉文字（全球最快推論速度）
- 智慧文字潤稿 — 移除贅字、修正文法、自動加標點
- 雙結果顯示 — 點擊「原文」或「潤稿」插入文字
- 支援繁體中文、英文、日文、自動偵測
- 中英混合語音支援
- 音檔逐字稿轉錄，可匯出 TXT、SRT
- 自帶金鑰（BYOK）— 使用自己的 API 金鑰，無需訂閱
- 隱私優先 — 不收集資料，音訊直接傳送至你的 API 服務商
- Material You 動態主題

**使用方式：**

1. 輸入你的 Groq API 金鑰（至 groq.com 免費申請）
2. 在系統設定中啟用「語墨語音」鍵盤
3. 在任何 App 中切換至語墨
4. 點擊麥克風按鈕開始說話
5. 點擊潤稿文字即可插入 — 完成！

**隱私：**

語墨採用自帶金鑰（BYOK）模式。你的音訊從裝置直接傳送至你選擇的 API 服務商。我們絕不會看到你的資料。
